使用 Docker 运行 MySQL

1. 快速启动 MySQL 容器

  • 使用 -p 3306:3306 将主机的 3306 端口映射到容器的 3306 端口。
  • 使用 --name mysql 为容器命名为 mysql。
  • 使用 -e MYSQL_ROOT_PASSWORD=123456 设置 MySQL 的 root 用户密码为 123456。
  • 使用 -d mysql:latest 后台运行最新版本的 MySQL。
